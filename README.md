# LUNA16-Lung-Nodule-Segmentation-U-Net-CBAM-FPN
A CPU-compatible lung nodule segmentation model based on U-Net+CBAM+FPN, with bug fixing, feature size alignment and valid mask generation. Implemented for medical image processing (PyTorch).（基于 U-Net+CBAM+FPN 的肺结节分割模型，CPU 适配，解决通道不匹配 / 尺寸对齐等 4 类核心问题，实现有效掩码生成与结果可视化，PyTorch 工程化实现）
